# Product Image Guidelines - Rudy's Store

## Recommended Image Specifications

### Primary Product Image

| Property          | Value                |
| ----------------- | -------------------- |
| **Dimensions**    | 1000×1000px (square) |
| **Minimum**       | 800×800px            |
| **Format**        | WebP or JPEG         |
| **Max File Size** | 200KB                |
| **Aspect Ratio**  | 1:1                  |

### Additional Gallery Images

- Same specifications as primary image
- Maximum 5 images per product recommended
- Show different angles, details, and usage

---

## Image Types by Category

### Clothing & Fashion

- **Main shot**: Product on white/neutral background
- **Detail shot**: Fabric texture, buttons, stitching
- **Lifestyle shot**: Model wearing the item

### Footwear (Slide & Sole)

- **Main shot**: Side profile view
- **Top view**: Show the shoe from above
- **Detail shot**: Sole, branding, material closeup

### Accessories (Rudy Luxury)

- **Main shot**: Product centered on clean background
- **Scale shot**: Show size context (e.g., worn on model)

---

## Best Practices

1. **Consistent Lighting** - Use soft, diffused light; avoid harsh shadows
2. **Clean Background** - White or neutral gray backgrounds work best
3. **Center the Product** - Keep product centered with padding around edges
4. **High Resolution** - Upload at recommended size for crisp display
5. **Optimize File Size** - Compress images using TinyPNG or Squoosh

- **Remove.bg** - https://remove.bg (background removal)
