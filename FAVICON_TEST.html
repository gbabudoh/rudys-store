Testing Instructions

Desktop Browsers

  1. Open this page in Chrome, Firefox, Safari, and Edge
  2. Check the browser tab - you should see the favicon
  3. Bookmark this page - check the bookmark icon
  4. Clear cache if favicon doesn't appear (Ctrl+Shift+Delete)
  5. Try hard refresh (Ctrl+F5 or Cmd+Shift+R)

Safari Pinned Tab (macOS)

  1. Right-click the tab
  2. Select "Pin Tab"
  3. The pinned tab should show a monochrome version of the icon

iOS (Safari)

  1. Open this page in Safari on iPhone/iPad
  2. Tap the Share button
  3. Select "Add to Home Screen"
  4. Check the icon on your home screen

Android (Chrome)

  1. Open this page in Chrome on Android
  2. Tap the menu (three dots)
  3. Select "Add to Home screen"
  4. Check the icon on your home screen

Troubleshooting

Favicon Not Showing?

  • Clear Cache: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
  • Hard Refresh: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
  • Incognito Mode: Test in private/incognito window
  • Check Console: Open DevTools (F12) and check for 404 errors
  • Wait: Sometimes browsers cache favicons for a while
