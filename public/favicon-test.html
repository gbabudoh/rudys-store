üí° Nuclear Option (If Nothing Works)

  1. Close Firefox completely
  2. Open File Explorer (Windows) or Finder (Mac)
  3. Navigate to:
    • Windows: %APPDATA%\Mozilla\Firefox\Profiles\
    • Mac: ~/Library/Application Support/Firefox/Profiles/
  4. Find your profile folder (ends with .default or .default-release)
  5. Delete the file: favicons.sqlite
  6. Restart Firefox
  7. Visit this page again
